# Инструкция для работы с Markdown

## Выделение текста

Для выделения текста курсивом нужно выделить текст звёздочкой (*)
Пример: *Вот так*
Для выделения текста полужирным нужно выделить текст двумя звёздочками (**)
Пример: **Вот так**
